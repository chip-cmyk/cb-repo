# Collapse 按钮

Collapse 组件用于折叠和展开内容，通常用于节省空间或隐藏不必要的信息。它可以包含多个面板，每个面板可以单独折叠或展开。

## 基础用法

使用 `v-model` 来控制折叠面板的展开和收起状态。可以通过 `accordion` 属性来设置是否只能展开一个面板。
